This plugin deemed to host general IDE functionality one could expect to find in
any/most MPS-based RCP application.

Think of this as a target to move [mps-workbench] stuff out of lib/mps-workbench.jar
into plugin world. There's mps-ide plugin to host functionality specific to
full-fledged MPS IDE and mps-devkit, with functionality targeted to Language Designers;
while this plugin is meant as a re-distribution piece to share some common IDE-like
functionality for any MPS user.

Indeed, it's a fine line to draw between rcp and ide, we just try to get started with it now.
